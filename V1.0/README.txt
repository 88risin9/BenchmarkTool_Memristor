这个benchmark tool V1.0实现了一个忆阻器阵列计算模拟器

此版本拥有 5 个功能

1. Modify Conductance Matrix (G) 修改电导G的值 后期可以影响忆阻器crossbar 的功耗

2. Modify Circuit Scaling  修改 外围电路 节点工艺 影响节点功耗（ADC TIA etc,) （一般化缩小）

3. Output Matrix Multiplication Result 输出矩阵向量运算结果

4. Output Power and Delay 输出 功耗 与 延时（面积？）

5. Simulate Neural Network(fully connected) （模拟全连接神经网络运算 并输出结果）
